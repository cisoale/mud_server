Realm Builder Help

Basic Controls

  • Left Click Room → Open editor
  • Drag Room → Move room
  • Drag Empty Space → Move map
  • ALT + Click Exit → Delete exit

Link Mode

  • Activate "Link Mode"
  • Click first room
  • Click second room
  • Exit is created automatically

Exit Colors

  • Gray → Normal Exit
  • Red → Locked Exit
  • Dashed → Secret Exit
